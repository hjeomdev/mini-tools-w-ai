# 유니코드 변환기

문자와 유니코드 코드포인트를 양방향으로 변환하는 도구.

## 레이아웃

- 입력칸(왼쪽) / 출력칸(오른쪽) 좌우 나란히 배치
- 두 칸의 스크롤이 동기화됨 (한쪽 스크롤 시 반대쪽도 동일 위치로 이동)
- **보기 모드** — 편집 보기(기본)와 행 단위 보기 중 선택
  - **편집 보기**: 두 textarea가 나란히 표시. 입력 가능
  - **행 단위 보기**: 줄 번호 | 입력 줄 | 출력 줄 형태의 테이블. git diff처럼 같은 행이 나란히 정렬됨

## 주요 기능

- **양방향 변환** — 상단 토글로 방향 전환 (문자 → 유니코드 / 유니코드 → 문자)
- **실시간 변환** — 입력 즉시 변환 결과 출력
- **다양한 유니코드 형식** — 출력 형식을 드롭다운으로 선택 (문자 → 유니코드 방향에서만 활성)
- **대/소문자 선택** — 16진수 형식에 한해 대문자/소문자 전환 (`U+` 형식은 항상 대문자 고정)
- **공백/개행 변환 여부** — 문자 → 유니코드 방향에서 공백·탭·개행을 변환할지 그대로 유지할지 선택 (기본값: 유지)
- **토큰 구분자** — 문자 → 유니코드 방향에서 변환된 토큰 사이에 공백을 넣을지 선택 (기본값: 공백 있음)
- **언어 감지** — 유니코드 → 문자 방향일 때 결과 문자의 언어/문자계열 자동 감지, 출현 빈도 내림차순으로 배지 표시
- **복사 버튼** — 입력칸·출력칸 각각 독립 복사 버튼
- **다크/라이트 모드** — `prefers-color-scheme` 자동 감지

## 유니코드 형식 (문자 → 유니코드)

| 형식 | 예시 (공백 있음) | 예시 (공백 없음) |
|------|-----------------|-----------------|
| `U+XXXX` | `U+AC00 U+B098` | `U+AC00U+B098` |
| `\uXXXX` | `가 나` | `가나` |
| `\u{XXXX}` | `\u{AC00} \u{B098}` | `\u{AC00}\u{B098}` |
| `&#xXXXX;` | `&#xAC00; &#xB098;` | `&#xAC00;&#xB098;` |
| `&#DDDDD;` | `&#44032; &#45208;` | `&#44032;&#45208;` |
| `0xXXXX` | `0xAC00 0xB098` | `0xAC000xB098` |
| 십진수 | `44032 45208` | `4403245208` |

- `&#DDDDD;`(HTML 10진수)·십진수 형식은 대소문자 옵션 비활성화
- `U+XXXX` 형식은 항상 대문자 고정

## 공백/개행 처리 (문자 → 유니코드)

| 설정 | 동작 |
|------|------|
| 유지 (기본) | 공백·탭·개행은 변환하지 않고 그대로 출력. 인접한 코드 토큰들은 구분자(공백/없음)로 구분 |
| 변환 | 모든 문자를 예외 없이 유니코드 토큰으로 변환 (공백 → `U+0020` 등) |

## 유니코드 → 문자 파싱 (자동감지 + 혼합 입력)

- 입력 텍스트에서 유니코드 토큰 패턴만 변환하고, 나머지 일반 문자는 그대로 유지
  - 예: `안녕 U+1F600 하세요` → `안녕 😀 하세요`
- 패턴 감지 우선순위:
  1. `U+XXXX`
  2. `\u{XXXX}`
  3. `\uXXXX`
  4. `&#xXXXX;`
  5. `&#DDDD;`
  6. `0xXXXX`
  7. 폴백: 패턴 없는 경우 입력 그대로 반환
