# 텍스트 비교기

두 텍스트를 줄 단위로 비교하고 차이를 시각화하는 도구.

## 주요 기능

- **실시간 비교** — 입력 즉시 80ms 디바운스 후 자동 비교
- **분할/통합 뷰** — 좌우 나란히(split) 또는 단일 스크롤(unified) 전환
- **인라인 문자 강조** — 변경된 줄 내에서 달라진 글자를 문자 단위로 하이라이트
- **변경 통계** — 추가/삭제/변경 줄 수 실시간 표시
- **저장 이력** — 수동 저장 및 3초 유휴 후 자동 저장. localStorage에 최대 50건 유지
- **이력 불러오기** — 저장된 이력에서 텍스트 쌍을 복원하여 재비교

## 행 유형

| 클래스 | 의미 |
|--------|------|
| `eq` | 동일 줄 |
| `rem` | 삭제된 줄 (A에만 존재) |
| `add` | 추가된 줄 (B에만 존재) |
| `chg-l` / `chg-r` | 변경된 줄의 이전/이후 (split 뷰에서 2행으로 분리) |

## 알고리즘

- **줄 단위 diff**: LCS(최장 공통 부분 수열), `Int32Array` 플랫 DP 테이블 사용
- **문자 단위 diff**: 변경 줄에 한해 동일 LCS 적용. 두 줄 합산 800자 초과 시 문자 강조 생략

## 데이터 저장

`localStorage` 키 `textdiff_history` (JSON 배열, 최대 50건).  
각 항목: `{ id, name, ts, a, b }`
